RedWeb Texture Loading Diagnostic

This page tests whether YouTube thumbnail textures load correctly in your browser. Open the browser console (F12) for detailed logs.

1. Raw Image Load Test

Tests if the browser can load each thumbnail URL directly as an <img> element.

2. WebGL Texture Load Test

Tests if THREE.js can load each URL as a GPU texture (tests CORS + WebGL texture upload).

3. UV Mapping Visualization

Shows how the texture SHOULD map to the card geometry (with the UV fix applied).
